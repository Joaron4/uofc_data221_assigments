## File description

The files are the questions 1 to 10 enumerated in the following way:

- question{number_of_question}.py

The files are in .py format extension.

Each file responds to the next questions:
1. Word Frequency Analysis - Compute and print the top 10 most frequent words from a text file.
2. Bigram Frequency Analysis - Analyze pairs of consecutive words and print the top 5 bigrams.
3. Near-Duplicate Line Detection - Identify lines that become identical after normalization.
4. Filtering Tabular Data - Filter student data based on criteria and save to a new file.
5. Categorical Variable and Summary - Create grade bands and generate grouped summaries.
6. Crime Risk Categorization - Classify areas by crime levels and compare unemployment.
7. Web Scraping Basics - Extract title and first paragraph from Wikipedia.
8. Extracting Headings - Scrape section headings from a Wikipedia page.
9. Tabular Data Extraction - Extract and structure table data from a webpage.
10. Text Search Function - Create a function to find lines containing a keyword.
